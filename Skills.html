My Skills

HTML 5

HTML is HyperText Markup Language, the standard and most common markup language for information
to be displayed through a web browser, such as a web page. A markup language is a system of symbols
that define the elements within a document, dictating its structure and formatting. Because of the wide use of HTML,
it remains one of the fundamental web design and development skills to know.

CSS

CSS—Cascading Style Sheets—is a style sheet language, a computer language that governs the visual appearance of web page elements.
CSS controls characteristics such as the layout of the elements, the colors that appear on the screen and the text font.
It's essential to the creation of websites since it controls the design components of the process.

Node JS

(Node) is an Open Source, cross-platform runtime environment for executing JavaScript code. Node is used extensively for server-side programming,
making it possible for developers to use JavaScript for client-side and server-side code without needing to learn an additional language.

React

(also known as React.js or ReactJS) is a free and open-source front-end JavaScript library that aims to make building user interfaces based on components more "seamless".

Python

Python programming languages. A programming language is a set of instructions, syntax and rules that instruct a computer
to perform specified tasks. It's what allows a web developer to create dynamic elements for a website.

JavaScript

is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.
